# Eigen Test - Angga Saeful

## Backend

API sistem peminjaman buku perpustakaan. ExpressJS + TypeScript, arsitektur DDD, SQLite, Swagger.

### Install & Run

```bash
cd backend
npm install
npm run dev
```

Server jalan di `http://localhost:3000`
Swagger docs di `http://localhost:3000/api-docs`

### Run Test

```bash
cd backend
npm test
```

## Algoritma

Solusi 4 soal algoritma menggunakan TypeScript.

### Install & Run

```bash
cd algoritma
npm install
npx ts-node src/soal1.ts
npx ts-node src/soal2.ts
npx ts-node src/soal3.ts
npx ts-node src/soal4.ts
```

### Run Test

```bash
cd algoritma
npm test
```
